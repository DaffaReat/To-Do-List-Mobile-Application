## Key Technical Decisions Made During Session

- **No auth** — per proposal, relies on OS-level file system encryption
- **Embedded HTML** — switched from `window.show(filepath)` to `window.show(HTML)`
  after WebUI2 failed to serve files from disk
- **`e.get_string()` / `e.return_string()`** — correct WebUI2 v2.5.7 event API,
  discovered by inspecting `dir(webui.event)` during debugging
- **Single shared SQLite connection** — WAL mode ensures safe concurrent reads
  and atomic writes even on force-close
